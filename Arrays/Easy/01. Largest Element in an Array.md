# Largest Element in an Array
### Brute Force : sort then last element
### Optimal  : take lar then compare
Code : 
```
class Compute {
    
    public int largest(int arr[], int n)
    {
        int largest=arr[0];
        for(int i=0;i<n;i++){
            if(arr[i]>largest){
                largest=arr[i];
            }
        }
        return largest;
    }
}
```
