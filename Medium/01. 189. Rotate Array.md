Rotate array by K elements https://takeuforward.org/data-structure/rotate-array-by-k-elements/

```
class Solution {
    public void rotate(int[] nums, int k) {
        int a[]=new int[nums.length];
        int n=nums.length;
        for(int i=0;i<n;i++){
            a[(i+k)%n]=nums[i];
        }
        for (int i = 0; i < nums.length; i++) {
            nums[i] = a[i];
        }
    }
}
```
